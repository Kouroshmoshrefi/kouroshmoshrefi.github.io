About me

Lets take a look at the road i went trough

  • At first

    I started from networks

    I've studied Network+ and networking protocols. i also learned a little about cisco in CCNA courses.

  • Then

    Linux, feel the freedom!

    when most of the servers are running on Linux, it's essential to know it and it's structure. how ever. i just started learning Linux just becouse it was fun! :)

  • So i started

    Python is begginer friendly

    I started learning programming by python. at first it helped me learn coding on procedural paradigm. Solving problems and learning algorithm and data structures

  • I choosed PHP!

    made for web

    I intrested in web development and what's better than a language witch was made to do so! i learned everything about functional and OOP paradigms, design patterns and principles. Architextures and more...

  • Database

    Mariadb / MySQL

    Storing data in a correct way is a must. relational databases keep it organized and easy!

  • Laravel

    For web artisans!

    There's no need for reinventing the wheel! by using a good framewok you can write codes faster and reduce the chance of any unwanted behaviour.

  • Time to have fun!

    JavaScript

    Started with core concepts, learned it and choosed to learn nodeJs also. i have to admit js is very fun!

  • Java

    Scale up!

    When it comes to performance no one can deny that the Java is more hardware efficient.
